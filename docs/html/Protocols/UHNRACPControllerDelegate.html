UHNRACPControllerDelegate Protocol Reference

Conforms to NSObject
Declared in UHNRACPControllerDelegate.h

Overview

The UHNRACPControllerDelegate protocol defines the methods that a delegate of a UHNRACPController object must adopt. The optional methods of the protocol allow the delegate to be notified upon completion of RACP operations.

– racpController:didSetNotificationStateRACP:

Notifies the delegate after a successful RACP operation

- (void)racpController:(id)controller didSetNotificationStateRACP:(BOOL)enabled

Parameters

controller

The UHNRACPController that initiated the RACP operation, which could be any of a number of different controller types, hence its type is ‘id’
enabled

The state that the RACP notification was set to

Discussion

This method is invoked after successful completion of an RACP operation
